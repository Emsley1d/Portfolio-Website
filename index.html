Project ReadMe

NUTRICO2

Project ReadMe

A CRUD application built over the course of a week as part of a pair. I worked on the basic functionality, the file/folder structure, user authentication and ultimately presented the project. The project was built using; HTML, CSS, Python, JSON, JavaScript, Django and NoSQL.

User's can sign up to the website; upload a photo of a recipe's ingredients; the photos are converted into text and the user provided with a list of Nutrional information and the CO2 emisions of the raw ingredients.

RATE MY CAT

Project ReadMe

A CRUD application built over the course of a week. This was a group project however, I focused on intergrating the Cat model and it's CRUD functionality, user image upload via Multer, app responsiveness and overall design. The project was built using; HTML, CSS, JavaScript, jQuery, Express (including CRUD and Authentication), API CRUD, Middleware, MongoDB and Mongoose.

Users can sign up to the website; upload photographs of their cats, rate other user's cats and view a leaderboard of the current ratings.
