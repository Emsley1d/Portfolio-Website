POKEGOTCHI

Project ReadMe

A fullstack MERN application built over the course of a week. I worked in a team of three and focused predominantly on the Front End; linking an external API with an Axios request; the overall design and user interface as well as presenting the project.

PokeGotchi is a Tamagotchi style app; users can sign up and select a Pokemon to become their 'PokeGotchi' pet. User's can then clean, feed, heal and play with their PokeGotchi to restore its HP (Hit Points).

NUTRICO2

Project ReadMe

A CRUD application built over the course of a week as part of a pair. I worked on the basic functionality, the file/folder structure, user authentication and ultimately presented the project. The project was built using; HTML, CSS, Python, JSON, JavaScript, Django and NoSQL.

User's can sign up to the website; upload a photo of a recipe's ingredients; the photos are converted into text and the user provided with a list of Nutritional information and the CO2 emissions of the raw ingredients.

RATE MY CAT

Project ReadMe

A CRUD application built over the course of a week. This was a group project however, I focused on intergrating the Cat model and it's CRUD functionality, user image upload via Multer, app responsiveness and overall design. The project was built using; HTML, CSS, JavaScript, jQuery, Express (including CRUD and Authentication), API CRUD, Middleware, MongoDB and Mongoose.

Users can sign up to the website; upload photographs of their cats, rate other user's cats and view a leaderboard of the current ratings.

ASTEROID BELT

Project ReadMe

A Space-Invaders style game built solely by myself over the course of a week using; HTML, CSS, JavaScript and jQuery.

Users can move a spaceship left and right whilst firing missiles in an attempt to destroy approaching Asteroids. User input is controlled by keypress events.
